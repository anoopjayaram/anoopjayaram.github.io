Does RPA kill Jobs?

Overview of Robotic Process Automation, Jan 7, 2022

RPA?

Robotic Process Automation, it is not like a Humanoid robot do all the tasks as seen in the movies! Robotic, a program that imitates the human actions. A Process is a work flow or sequence of steps that’s lead to an activity or a function. Automation is any process that is done by a robot without human intervention. It’s a tool or software robot that do task without human intervention. You may wonder when you login to a portal, there is a question like Are you a robot?

In some websites , they ask to type some words or numbers inside a box or images names as CAPTCHA (Completely Automated Public Turing test to tell Computers and Human Apart). It is implemented to block spammers and software bots that are try to automatically create email accounts, sign up automatically, access databases etc.

main objective of the CAPTCHA is to make sure the resource is accessing by the human, not a bot. The updation of CAPTCHA, reCAPTCHA ensures more security towards bots. This test is almost impossible to solve by the software robots. It’s about automating the manual jobs as repetitive task based on a pattern like combining the spreadsheet documents from different sources to a single one, scrapping (collecting) data from online resources, bank tellers, etc. As part of industrial automation, many labours were replaced by machines. The same issues arrived in history before when the computer was introduced. Introducing technology doesn’t make peoples jobless. It creates new opportunities and time to skill up.

Working

The RPA tools have the ability to imitate the user action, especially repetitive tasks. We have to set the steps or work flow using RPA tool. Once the pattern is created, it will work continuously without any mistakes till the project is cancelled. The tools like Blue Prism, Ui Path, Automation Anywhere etc are used by many companies as well as developers. Some of these tools have free or trial versions. We can learn the basics and implement it.
